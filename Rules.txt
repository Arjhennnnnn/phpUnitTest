Bowling Rules

- The Goal is to knock down all ten pins

- Each frame consists of throwing 
    the ball twice to knock down all the pins

- If you knock down all the pins with the first ball , 
    it is called a "strike"

- IF you knock down all the pins with the second ball,
    it is called a spare

- Each games consists of ten frames, 
    
    - If you bowl a strike in the tenth frame, 
    you get two more balls

    - If you throw a spare , you get one more ball

- Scoring is based on the number of pins you knock down



A game
consists of 10 frames
each frames has 1-2 rolls
